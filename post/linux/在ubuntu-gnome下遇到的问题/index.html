修改gnome主题的各种配置

到目录/usr/share/gnome-shell/theme,修改g3d.css(若无此文件尝试修改其他.css文件)

1

对于Vivado缺乏库的问题

application-specific initialization failed: couldn't load file "librdi_commontasks.so": libtinfo.so.5: cannot open shared object file: No such file or directory

解决方法:

1

sudo apt install libncurses5

直接将命令行输出复制到剪贴板

1
2
3
4

sudo apt install xclip
alias toc='xclip -selection c'
##实例:直接将apple.txt的文件输出到剪贴板
cat apple.txt | toc

为git常用命令定义别名

1
2
3
4
5
6

alias ga='git add -A'
alias gm='git commit -m'
alias gam='git add -A;git commit -m'
alias gamn='git add -A;git commit -m "nothing"'
alias gp='git push'
alias gamnp='git add -A;git commit -m "nothing";git push'

Ubuntu安全重启

按住ctrl和alt,然后依次SysRq, R, E, I, S, U, B.

删除使用python3 setup.py install安装的文件

1
2

cat files.txt | xargs rm -rf
