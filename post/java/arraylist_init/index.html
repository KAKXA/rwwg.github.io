Java8中ArrayList在一条语句中完成初始化的4种方法

2021-09-23

Contents

    • Arrays工具类
    • Collections工具类
    • 匿名内部类
    • JDK8 Stream

ArrayList的初始化是经常会遇到的情形,这里总结了4种快速在初始化时赋值的方法

Arrays工具类

使用Arrays.asList,返回一个定义在List类中的不可变的ArrayList,从而需要用java.util.ArrayList包装一下.

1

List<String> jdks = new ArrayList<>(Arrays.asList("jdk1", "jkd2"));

Collections工具类

使用Collections.nCopies,可以构造多个重复元素构成的List,其也返回一个不可变的ArrayList,需要用java.util.ArrayList包装一下.

1

List<String> apples = new ArrayList<>(Collections.nCopies(3, "apple"));

匿名内部类

下式中,有两个花括号:

  1. 外侧的大括号定义了一个继承于ArrayList的匿名内部类
  2. 内侧的大括号是这个匿名内部类的实例初始化块(非静态初始化块)

等号右侧new了这个ArrayList的子类的实例并**向上转型为ArrayList的引用

1
2
3
4
5

List<String> names = new ArrayList<String>() {{
    add("Tom");
    add("Sally");
    add("John");
}};

关于匿名内部类的详细内容,请看Java内部类

JDK8 Stream

Collectors类实现了很多归约操作,比如将流转换成集合和聚合元素

1

List<String> colors = Stream.of("blue", "red", "yellow").collect(Collectors.toList());
