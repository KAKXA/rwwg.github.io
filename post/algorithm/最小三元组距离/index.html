最小三元组距离

2021-03-26

Contents

      • 题目
      • 思路
      • 代码

主要是为了记录一个神奇的公式.

题目

已知三个升序整数数组a[l], b[m]和c[n].请在三个数组中各找一个元素,使得组成的三元组距离最小.

三元组的距离定义是:假设a[i],b[j]和c[k]是一个三元组,那么距离为:Distance = max(|a[i]–b[j]|,|a[i]–c[k]|,|b[j]–c[k]|)请设计一个求最小三元组距离的最优算法,并分析时间复杂度.

思路

神奇的公式: $$ max(|x_1-x_2|,|y_1-y_2|) =\frac{|x_1+y_1-x_2-y_2|+|x_1-y_1-(x_2-y_2)|}{2} $$ 则可以推出: $$ max(|a[i]–b[j]|,|a[i]–c[k]|,|b[j]–c[k]|) = \frac{|a[i]-b[j]|+|a[i]-c[k]|+|b[j]-c[k]|}{2} $$ 要使等式右侧分子最小化,用三个指针同时指向a[0],b[0],c[0],每次计算一次上式,然后移动值最小的指针.

终止条件是任何一个数组被遍历完.时间复杂度是 $O(l + m + n)$

def f(a, b, c):
    def cal(a, b, c):
        return abs(a - b) + abs(a - c) + abs(b - c)
    l, m, n = len(a), len(b), len(c)
    res = float('inf')
    i, j, k = 0, 0, 0
    while i < l and j < m and k < n:
        res = min(res, cal(a[i], b[j], c[k]))
        if a[i] <= b[j] and a[i] <= c[k]:
            i += 1
        elif b[j] <= a[i] and b[j] <= c[k]:
            j += 1
        else:
            k += 1
    return res
