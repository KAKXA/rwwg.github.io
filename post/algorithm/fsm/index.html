FSM(有限状态机),在算法题中,适合处理复杂的输入字符串等情况.

FSM的概念

有限个状态以及在这些状态之间的转移和动作行为的数学模型

FSM的三特点

  1. 一个时刻,只有一个状态
  2. 某种条件下,会从一种状态转变到另一种状态
  3. 状态有限

FSM的四要素

  1. currentState: 当前状态
  2. nextState: 下一个状态
  3. transition(currentState, action): 当前状态和下个状态的关系
  4. action: state变动的event

FSM状态转换图

  • 使用状态转换图描述FSM
  • 状态转换图中的节点对应不同的状态对象
  • 每个状态对象通过一个输入字符转换到另一个状态上,或者保持原状态不变

FSM模板题

题目 1

[8] 字符串转换整数 (atoi) (少壮不努力,老大atoi)

请你来实现一个 myAtoi(string s) 函数，使其能将字符串转换成一个 32 位有符号整数（类似 C/C++ 中的 atoi 函数）。

函数 myAtoi(string s) 的算法如下：

读入字符串并丢弃无用的前导空格
检查下一个字符（假设还未到字符末尾）为正还是负号，读取该字符（如果有）。 确定最终结果是负数还是正数。 如果两者都不存在，则假定结果为正。
读入下一个字符，直到到达下一个非数字字符或到达输入的结尾。字符串的其余部分将被忽略。
将前面步骤读入的这些数字转换为整数（即，"123" -> 123， "0032" -> 32）。如果没有读入数字，则整数为 0 。必要时更改符号（从步骤 2 开始）。
如果整数数超过 32 位有符号整数范围 [−231,  231 − 1] ，需要截断这个整数，使其保持在这个范围内。具体来说，小于 −231 的整数应该被固定为 −231 ，大于 231 − 1 的整数应该被固定为 231 − 1 。
返回整数作为最终结果。

注意：

本题中的空白字符只包括空格字符 ' ' 。
除前导空格或数字后的其余字符串外，请勿忽略 任何其他字符。

in_number end end in_number end

import java.util.HashMap;
import java.util.Map;

class FSM {
    public String state = "start";
    public int sign = 1;
    public long res = 0;
    public Map<String, String[]> transitions = new HashMap<>();
    public FSM() {
        transitions.put("start", new String[] {"start", "sign", "number", "finish"});
        transitions.put("sign", new String[] {"finish", "finish", "number", "finish"});
        transitions.put("number", new String[] {"finish", "finish", "number", "finish"});
        transitions.put("finish", new String[] {"finish", "finish", "finish", "finish"});
    }

    public int get_col(char c) {
        // 通过字符获取其属于数组中哪一列的情况
        if (c == ' ') {
            return 0;
        } else if (c == '+' || c == '-') {
            return 1;
        } else if (Character.isDigit(c)) {
            return 2;
        } else {
            return 3;
        }
    }

    public void get(char c) {
        // 根据字符c进行状态变化
        state = transitions.get(state)[get_col(c)];
        if (state.equals("number")) {
            res = res * 10 + c - '0';
            res = sign == 1 ? Math.min(Integer.MAX_VALUE, res) : Math.min(-(long)Integer.MIN_VALUE, res);
        } else if (state.equals("sign")) {
            if (c == '+') {
                sign = 1;
            } else {
                sign = -1;
            }
        }
    }
}


class Solution {
    public int myAtoi(String str) {
        FSM fsm = new FSM();
        for(char c: str.toCharArray()) {
            fsm.get(c);
        }
        return (int)(fsm.sign * fsm.res);
    }
}
