430. 扁平化多级双向链表

这题的难点在于不容易考虑清楚迭代时链表之间的跳跃要如何处理

考虑从交界节点跳到另一链表,处理完子节点,返回子节点的最后一个节点时

  1. 如果交界节点pre是其链表的最后节点,那么,最后返回的应该是其子节点中的最后一个节点
  2. 否则,last应该在pre后面诞生,步骤中的last应该是pre.在真正处理时,只要没有子节点的节点,我们都把pre赋给它

// Definition for a Node.
class Node {
    public int val;
    public Node prev;
    public Node next;
    public Node child;
};
*/

class Solution {
    public Node flatten(Node head) {
        dfs(head);
        return head;
    }

    public Node dfs(Node node) {
        Node tmp = node;
        // 记录链表的最后一个节点
        Node last = null;

        while (tmp != null) {
            // pre指向交界节点
            Node pre = tmp;
            tmp = tmp.next;
            //  如果有子节点，那么首先处理子节点
            if (pre.child != null) {
                Node childLast = dfs(pre.child);

                //  将 node 与 child 相连
                pre.next = pre.child;
                pre.child.prev = pre;

                //  如果 next 不为空，就将 last 与 next 相连
                if (tmp != null) {
                    childLast.next = tmp;
                    tmp.prev = childLast;
                }

                // 将 child 置为空
                pre.child = null;

                // 难点在于这里last在两种不同情况下的细节处理
                // 如果交界节点pre是其链表的最后节点,那么,最后返回的应该是其子节点中的最后一个节点
                last = childLast;
            } else {
                // 难点在于这里last在两种不同情况下的细节处理
                // 只要没有子节点的节点,我们都把pre赋给它
                last = pre;
            }
        }
        return last;
    }
}
