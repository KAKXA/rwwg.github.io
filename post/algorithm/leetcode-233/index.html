数字 1 的个数

问题描述:

给定一个整数 n,计算所有小于等于 n 的非负整数中数字 1 出现的个数.

思路

  1. 对于某一位,设其位权为$10 ^ i$,$i$是该位从右往左数的位次.
  2. 以这一位为界将数分成$higher$,$lower$两部分
  3. 该位上一共出现的1的个数为
    • $higher \times divider$ 若该位为0;
    • $higher \times divider + lower + 1$ 若该位为1
    • $higher \times divider + divider$ 若该位大于1

class Solution:
    def countDigitOne(self, n: int) -> int:
        res = 0
        divider = 1
        while True:
            higher = n // (divider * 10)
            mid = (n // divider) % 10
            lower = n % divider

            if mid == 0:
                res += higher * divider
            elif mid == 1:
                res += higher * divider + (lower + 1)
            else:
                res += higher * divider + divider

            if higher == 0:
                break
            divider *= 10
        return res
