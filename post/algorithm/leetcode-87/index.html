扰乱字符串

题目描述

使用下面描述的算法可以扰乱字符串 s 得到字符串 t ：

  1. 如果字符串的长度为 1 ，算法停止
  2. 如果字符串的长度 > 1 ，执行下述步骤：
    • 在一个随机下标处将字符串分割成两个非空的子字符串。即，如果已知字符串 s ，则可以将其分成两个子字符串 x 和 y ，且满足 s = x + y 。
    • 随机 决定是要「交换两个子字符串」还是要「保持这两个子字符串的顺序不变」。即，在执行这一步骤之后，s 可能是 s = x + y 或者 s = y + x 。
    • 在 x 和 y 这两个子字符串上继续从步骤 1 开始递归执行此算法。

给你两个 长度相等 的字符串 s1 和 s2，判断 s2 是否是 s1 的扰乱字符串。如果是，返回 true ；否则，返回 false 。

思路

三维动态规划.dp[i][j][k] 表示s1从i下标开始长为k的子串与s2从j下标开始长为k的子串是否互为扰乱字符串.

转移方程是利用这样的思路,对于这两个长为k的子串a,b,考虑从第1个,第2个,…,第k个元素前面的位置拆分将其拆为两个字符串,如果任意一次拆分使得这种拆分得到的4个子串两两对应的dp数组位置的值为真,则子串a,b是互为扰乱字符串的.注意,由于扰乱字符串是前一个字串可以变动到后面,所以要另外考虑前后子串交换的情况.

class Solution:
    def isScramble(self, s1: str, s2: str) -> bool:
        n = len(s1)
        dp = [[[False] * (n + 1) for _ in range(n)] for _ in range(n)]
        # 长度为1时,两个下标处是否相同
        for i in range(n):
            for j in range(n):
                dp[i][j][1] = s1[i] == s2[j]
        # 长度为1时已经处理了,现在从长度为2开始处理
        for le in range(2, n + 1):
            # 把s1中起点限制在下标不会越界的位置
            for i in range(n - le + 1):
                # 把s2中起点限制在下标不会越界的位置
                for j in range(n - le + 1):
                    # 枚举小于le的所有子串长度
                    # 两个子串长度分别为k和le-k
                    for k in range(1, le):
                        if dp[i][j][k] and dp[i + k][j + k][(le - k)]:
                            dp[i][j][le] = True
                            break
                        if dp[i][j + (le - k)][k] and dp[i + k][j][(le - k)]:
                            dp[i][j][le] = True
                            break
        return dp[0][0][n]
