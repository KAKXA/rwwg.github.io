二叉搜索树迭代器

题目

实现一个二叉搜索树迭代器类BSTIterator ，表示一个按中序遍历二叉搜索树（BST）的迭代器：

BSTIterator(TreeNode root) 初始化 BSTIterator 类的一个对象。BST 的根结点 root 会作为构造函数的一部分给出。指针应初始化为一个不存在于 BST 中的数字，且该数字小于 BST 中的任何元素。
boolean hasNext() 如果向指针右侧遍历存在数字，则返回 true ；否则返回 false 。
int next()将指针向右移动，然后返回指针处的数字。

思路

考虑到中序遍历中,在结束一个叶结点的遍历后,要找到离该叶结点最近且右结点不为空的结点,这样才能进入一个右子树.故使用一个栈来维护当前结点到根结点的路径上未被使用过的结点,每当一个结点被next返回,就把它从栈中弹出,并把它的右子树入栈.

class BSTIterator:

    def __init__(self, root: TreeNode):
        self.cur = root
        self.arr = []

    def next(self) -> int:
        while self.cur:
            self.arr.append(self.cur)
            self.cur = self.cur.left

        cur = self.arr.pop()
        self.cur = cur.right
        return cur.val

    def hasNext(self) -> bool:
        return self.cur != None or len(self.arr) != 0
