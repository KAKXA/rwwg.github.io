从前序与中序遍历序列构造二叉树

题目

根据一棵树的前序遍历与中序遍历构造二叉树.

注意: 你可以假设树中没有重复的元素.

思路

用前序遍历找到当前的根节点,用后序遍历确定左右结点的总结点个数.这样就把一个大的前序,中序数组拆成两个小的前序,中序数组.并且是一一对应的.

class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> TreeNode:
        def dfs(prel, prer, inl, inr):
            # 用prel, prer规定前序遍历数组的左右边界
            # 用inl, inr规定中序遍历数组的左右边界
            # 当prel越过prer,意味着到达空结点了
            if prel > prer:
                return None
            preroot = prel
            inindex = m[preorder[preroot]]
            root = TreeNode(preorder[preroot])
            leftsize = inindex - inl
            rightsize = inr - inindex
            root.left = dfs(prel + 1, prel + leftsize, inl, inindex - 1)
            root.right = dfs(prel + leftsize + 1, prer, inindex + 1, inr)
            return root

        m = dict()
        l = len(inorder)
        for i in range(l):
            m[inorder[i]] = i
        return dfs(0, l - 1, 0, l - 1)
