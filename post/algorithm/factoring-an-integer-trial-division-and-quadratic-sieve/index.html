2. Trial Division

  • As the integer N includes 38 decimal digits, to find the factors with few digits, like factors fewer than 1,00,000,000, we use trial division. Iterating from 2 to 1,00,000,000, find integer n which meets the following equation: $$ N\equiv0\pmod n $$ When we find an n, record it and divide N by n to avoid appearing a composite number in the following iteration. After the loop meets its end, we check if M is equal to 1 or a prime number. For our M , it is divided into a integer with 27 decimal digits. Thus we should try better algorithm to solve the problem.

3. Quadratic Sieve

Fermat And Kraitchik

  • Before introducing quadratic sieve, consider the following question: find two factors of 9991. A clever way is to write 9991 as 10000 - 9, which is: $$ 9991=100^2-3^2=(100+3)\times(100-3)=103\times97 $$ so we find two factors 103 and 97. This method is a factorization of Fermat[1].

  • Now what we need is to find two square integers. It is hard to find numbers meeting the equation most of the time, and the method can be improved. In 1920s, Maurice Kraitchik[1] came up with an enhancement, consider: $$ a^2-b^2\equiv0\pmod N\ (a+b)(a-b)\equiv0\pmod N\ N|(a+b)(a-b) $$ (a+b) and (a-b) is respectively by at least one of N’s prime factors only if

    $$ \gcd(a+b,N)\ne1 $$ and $$ \gcd(a-b,N)\ne1 $$ satisfying the above conditions, gcd(a+b,N) and gcd(a-b,N) will be two factors of N.

Q(x)

  • Consider $$ x_{i}\equiv a_{i}^2\pmod N\ \ \ \ \ \ \ \ a_{i} = [\sqrt{N}]+i \ \ \ \ \ i = 1,2,3,4,… $$ It can be verified by computer that it is hard to find an x which is a square number in finite time. Now postulate that: $$ x_i\times x_j= t^2\ \ \ \ \ \ \ t\in \mathbb{Z} \ x_{i} \equiv a_{i}^2\pmod N\ x_{j} \equiv a_{j}^2\pmod N\ i\ne j $$ Now, although the two numbers are not square numbers, their multiplication is. So: $$ x_i\times x_j \equiv t^2 \pmod N $$ the left formula is also a square number so we can consider the two formula for the factors. In this way, we try to find more than one integer x. Define Q(x) as follows: $$ Q(x) \equiv ([\sqrt{N}] + x)^2 \pmod N $$

Y-Smooth

  • To find a series of Q(x) whose multiplication is a square number, we should construct a set of prime numbers. We define a vector Y, including several prime numbers. If Q(x) can only be divided by some elements of Y, we call it a Y-Smooth integer. We add Y-Smooth Q(x) into a vector Q. So $$ Q = (Q(x_1),Q(x_2),…,Q(x_n)) $$ At the same time, for each one in Q, present it as a multiplication of integers in Y, like: $$ Y=(2,3,5)\ Q(x)=300 = 2^2\times3^1\times5^2 $$ now develop a vector for Q(x) itself: $$ (2,1,2) $$ as 2’s and 5’s exponents are even numbers, we don’t need to consider them, since we only need an extra 3 to construct a square number 900. So the vector for Q(x) and Y convert to: $$ (0,1,0) $$

  • To get appropriate Y, consider: $$ ([\sqrt{N}] + x)^2 \equiv Q(x) \pmod N\ (\frac {Q(x)}{N})=1\ (\frac{p_1p_2…p_n}{N})= 1\ $$ If each p satisfies: $$ (\frac {p_i} N) = 1 $$ their product should be a square number. Constructing a proper Y helps to find Q(x) more easily.

System of Linear Congruence Equations

  • Let us change every Q(x) in Q to Q(x)’s vector presentation on Y. Thus, if the dimension of Y is y, the dimension of Q, the number of Y-smooth Q(x) that are found, is q, we construct a y*q matrix A[2]**. For example, for 6,8,9,12 and Y(2,3), the matrix A is like:

    $$ \begin{pmatrix} 1 & 1 & 0 & 0 \ 1 & 0 & 0 & 1 \end{pmatrix}\ $$ while q is greater than y, just like the above matrix A indicates, there must be at least a vector x which satisfies: $$ A_{y\times q}x_{q\times 1}\equiv0\pmod 2\ x = (i_1,i_2,i_3,…,i_q)^T\ \ \ \ \ \ i \in{0,1} $$ as for its proof, consider: $$ Ax = 0 $$ according to linear algebra, we know that, if rank(A) is smaller than n, which is the number of A’s column vectors, there will be (rank(A) - n) vectors which represent the system of polynomial solution. Then: $$ Ax \bmod 2=0\bmod 2\ A\times(x\bmod2)=0\bmod2\ Ax\equiv0\pmod2\ \ \ \ \ \ \ x = (i_1,i_2,i_3,…,i_q)^T\ \ \ \ \ \ i \in{0,1} $$ now solve the matrix A: $$ {\begin{pmatrix} 1 & 1 & 0 & 0 \ 1 & 0 & 0 & 1 \end{pmatrix}\} \begin{pmatrix} x_1\x_2\x_3\x_4\end{pmatrix} \equiv \begin{pmatrix} 0\0\0\0\end{pmatrix}\pmod2\ \begin{pmatrix} x_1\x_2\x_3\x_4\end{pmatrix}\equiv \begin{pmatrix}0\0\1\0\end{pmatrix}\pmod2\ or\ \begin{pmatrix} x_1\x_2\x_3\x_4\end{pmatrix}\equiv \begin{pmatrix} 1\1\0\1\end{pmatrix}\pmod2 $$

  • Try the second solution as an example. Because the 1st, 2nd, 4th position is 1, we select 6, 8 and 12. Their product is 576, a square number.
