有序链表合并

2021-03-29

Contents

有序链表合并是链表操作的基本功之一,类似的思想也是归并排序的基本组件.

这是思想很简单但是有些细节要仔细处理的算法,类似二分法.一旦用到必须熟练地写出来.

def mergeTwoLists(self, l1, l2):
    prehead = ListNode(-1)

    prev = prehead
    while l1 and l2:
        if l1.val <= l2.val:
            prev.next = l1
            l1 = l1.next
        else:
            prev.next = l2
            l2 = l2.next            
        prev = prev.next

    # 合并后 l1 和 l2 最多只有一个还未被合并完，我们直接将链表末尾指向未合并完的链表即可
    prev.next = l1 if l1 is not None else l2

    return prehead.next
