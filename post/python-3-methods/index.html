Python的实例方法,类方法与静态方法

2021-03-20

Contents

在复杂的数据结构中会会用到Python的类方法和静态方法.

  1. 实例方法:
1

class Example:
	def __init__(self):
		pass
	def hello(self, x):
		print(x)
example = Example()
example.hello('hello')

最常见的是实例方法,只能从实例调用,self就是实例这个参数.

class Example:
	count = 0
	def __init__(self):
		Example.count += 1
	@classmethod
	def get_count(cls):
		return cls.count

example1 = Example()
example2 = Example()

print(Example.get_count())
# 2
print(example1.get_count())
# 2

既可以用实例也可以用类调用,返回类变量(而非类的实例中的变量)
