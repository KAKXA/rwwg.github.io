Droidcam在Linux下的安装与使用

2021-04-05

Contents

Droidcam 是一个可以将手机摄像头作为电脑摄像头使用的工具,同时支持局域网和USB ADB调试.

电脑自带摄像头无法使用的情况下,可以用这个应用解燃眉之急.

这里只提供在Linux(KUbuntu)`下的安装以及对于Android手机的连接.

  1. 运行

    1

    lsmod | grep v4l2loopback_dc

    如果有输出,进行下一步,否则:

    1

    sudo apt install v4l2loopback-utils

    如果你的电脑有 Secure-boot 的设定,那么按照提示,你会设置一个Boot口令,重启电脑,并在Boot界面选择类似于 enroll MOK key 的一项,输入Boot口令.假如你第一次不知道怎么选并直接开机了,可以在terminal中输入

    1

    sudo update-secureboot-policy --enroll-key

    再进行一次重启,重复以上步骤.如果不在Boot时输入Boot口令,这个第三方模块就不会加载到内核中(大概,具体的不太了解).

  2. 到Droidcam的Linux安装界面,按照步骤安装即可.

  3. 启动Droidcam,并选择USB(Android).假如出现提示 v4l2loopback not Found 的信息,说明第一步中的步骤没有完成.

  4. 手机上安装Droidcam.

  5. 手机选择开发者模式,启用调试.

  6. 用USB连接手机和电脑,手机上会询问是否运行该IP地址对手机ADB调试,选择同意.

  7. 在电脑端Droidcam点击Connect.
