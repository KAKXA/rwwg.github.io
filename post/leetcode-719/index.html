leetcode-719: find k-th smallest pair distance

找出第 k 小的距离对

问题描述

给定一个整数数组,返回所有数对之间的第 k 个最小距离.一对 (A, B) 的距离被定义为 A 和 B 之间的绝对差值.

思路

由于第k小的距离限制在[0, max(num) - min(num)]之间,所以可以对这个区间进行二分.

  1. 先对数组排序;
  2. 用possible(guess),数出nums中所有距离小于等于guess的数对的个数,如果大于等于k,说明guess选大了,导致更多的数对被选中,在二分法中减少guess的值继续尝试.

class Solution:
    def smallestDistancePair(self, nums: List[int], k: int) -> int:
        def possible(guess):
            count = left = 0
            for right, x in enumerate(nums):
                while x - nums[left] > guess:
                    left += 1
                count += right - left
            return count >= k
        nums.sort()
        l = 0
        r = nums[-1] - nums[0]
        while l < r:
            mid = l + (r - l) // 2
            if possible(mid):
                r = mid
            else:
                l = mid + 1
        return l
