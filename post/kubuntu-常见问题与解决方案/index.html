解决fcitx-googlepinyin 不能在kde qt中使用的问题

在/etc/environment中添加

1
2
3

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx

重启.

Hibernate

  1. 编辑/etc/default/grub’
  2. GRUB_CMDLINE_LINUX_DEFAULT="quiet splash” 改为 GRUB_CMDLINE_LINUX_DEFAULT="quiet splash resume=/dev/sdXY(SWAP 分区)”
  3. sudo update-grub
  4. sudo update-initramfs -u
  5. systemctl hibernate 或 pm-hibernate
