块级元素,内联元素与空元素

块级元素

  • 其前面,后面的内容会被挤到下一行展现.
  • 块级元素通常用于展示页面上结构化的内容
    • 段落,列表,导航菜单,页脚
    • 块级元素不能被嵌套进内联元素中

内联元素

  • 内联元素不会导致文本换行
  • <a> <em> <strong>

空元素

  • 没有结束标签和内容,如<img>
