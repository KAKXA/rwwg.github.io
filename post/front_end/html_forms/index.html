HTML 表单是什么

HTML表单是用户和web站点或应用程序之间交互的主要内容之一。它们允许用户将数据发送到web站点。大多数情况下，数据被发送到web服务器，但是web页面也可以自己拦截它并使用它。

接下来将实现一个简单的联系人表单,包括收件人姓名,收件人E-mail,要发送的信息以及"发送信息"的按钮.

<form> 元素

1
2
3

<form action="/my-handling-form-page" method="post">

</form>

其中

<action> 定义了在提交表单时,应该把所收集的数据送给谁去处理

method 属性定义了发送数据的HTTP方法(get/post)

<form action="/my-handling-form-page" method="post">
  <div>
    <!-- 在所有<label>元素上使用for属性,它是将标签链接到表单小部件的一种正规方式,这个属性引用到对应的小部件的id,比如下面这行中的name就对应着下面的id="name" -->
    <label for="name">Name:</label>
    <!-- input中的type属性定义了<input>属性的行为方式,它可以从根本上改变元素,这里定义为"text",接受任何类型的文本输入 -->
    <!-- 如果要定义input的默认值,可以用:

    <input type="text" value="by default this element is filled with this text"/>
	
	-->
    <input type="text" id="name">
  </div>
  <div>
    <label for="mail">E-mail:</label>
    <!-- 这里定义了只接受格式正确的电子邮件地址的单行文本字段 -->
    <input type="email" id="mail">
  </div>
  <div>
    <label for="msg">Message:</label>
    <!-- 与上面的<input>标签不同,textarea标签不是空元素,因此必须关闭它
	如果要定义<textarea>的默认值,只需在<textarea>的开始和结束之间放置默认值:
	
	<textarea>by default this element is filled with this text</textarea>
	
	-->
    <textarea id="msg"></textarea>
  </div>
</form>

<div class="button"> 
  <button type="submit">Send your message</button>
</div>

type有三个可选值,分别是submit,reset,button.

  • submit: 发送表单的数据到<form>元素的action 属性所定义的网页
  • reset: 将所有表单小部件重新设置为它们的默认值
  • buttone: 不会发生任何事
