132 模式

题目

给你一个整数数组 nums ，数组中共有 n 个整数。132 模式的子序列 由三个整数 nums[i]、nums[j] 和 nums[k] 组成，并同时满足：i < j < k 和 nums[i] < nums[k] < nums[j] 。

如果 nums 中存在 132 模式的子序列 ，返回 true ；否则，返回 false 。

思路

  • 从右往左遍历数组,维护一个"2"的候选单调栈(栈顶到栈底依次递增), “2” 的候选值max_2.

  • 当遇到一个数时,检查它是不是大于栈顶元素,如果是,则它可以作为"3"的候选值,此时栈顶元素作为"2"的候选值max_2,不断迭代弹栈直到找到最大的"2"的候选值max_2.如果发现栈中没有元素比当前元素大,则把它放到栈底作为未来可能的"2"的候选值,这是因为它一定大于当前的max_2,所以对于左侧可能的更大的3,它都可以作为比当前 max_2 更好的"2"的候选值.

class Solution:
    def find132pattern(self, nums: List[int]) -> bool:
        n = len(nums)
        candidate_k = [nums[n - 1]]
        max_k = float("-inf")

        for i in range(n - 2, -1, -1):
            if nums[i] < max_k:
                return True
            while candidate_k and nums[i] > candidate_k[-1]:
                max_k = candidate_k[-1]
                candidate_k.pop     ()
            if nums[i] > max_k:
                candidate_k.append(nums[i])

        return False
